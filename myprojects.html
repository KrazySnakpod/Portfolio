My Projects

A list of all the projects I've worked on, from Game Jam submissions to University projects, as well as personal projects that I'm still working on.

Game Jam submissions

Clownery

Created during the Global Game Jam 2024 with the theme of "Make Me Laugh", Clownery is a multiplayer game with two main phases - a daytime phase, where the players perform on stage and use skills to gather money, and a nighttime phase, where players must stealthily break into a series of secured areas and steal as much as possible, using their money to purchase useful tools to simplify the process.

  • Created in Unity
  • Written in C#

The Traveller

Created during the Ryan Laley Games Summer Jam 2023 with the theme of "Contrast", The Traveller is a short game in which you play as a time-hopping traveller attempting to breach the castle. The traveller cannot attack, and as such must utilize their time-hopping abilities to manoeuvre past any enemies they encounter.

  • Created in UE5
  • Written in Blueprint

University projects

Survival Horror AI

A short demo created as part of my dissertation project, utilizing a GOAP system and the A* pathfinding algorithm to determine a course of events for each enemy. The player navigates a dark area while the enemies navigate the area, responding to sight and audio cues.

  • Created in UE5
  • Written in Blueprint & C++

Personal projects

Unfinished Turn-Based RPG

Inspired by numerous turn-based RPG games, I wanted to find out how the combat system in these games could be implemented. The player is able to use a basic attack, use skills costing HP or MP, and inflict status ailments that deal damage over time. Each entity has their own stat table, determining stats such as health, physical attack and speed dependent on their current level. This project started in Summer 2023 as my first Unreal Engine project.

  • Created in UE5
  • Written in Blueprint
